# SRS & Risk File Review Copilot - Product Requirements Document

## Status
Draft v1.1 - Portfolio MVP

## Overview
This PRD defines the requirements for an AI-assisted SRS and Risk File Review Copilot integrated into Git/GitHub workflows.

## Domain
Software as a Medical Device (SaMD) / medical device software - EU MDR, FDA, ISO 13485, IEC 62304, ISO 14971, IEC 81001-5-1, OWASP.

## Purpose
Automatically review version-controlled SRS and risk files at key lifecycle states, generating traceable findings and proposed corrections.
